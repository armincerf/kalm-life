Kalm is an unfinished suite of tools designed to help you lead a more relaxed life.

Kalm Routines

Kalm Routines is an iOS and android app that helps you stick to a list of tasks. You can define complex routines with multiple concurrent timed events and receive notifications to help keep you on track.
